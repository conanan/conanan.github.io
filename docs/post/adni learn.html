ADNI 研究相关说明

  • ADNI（阿尔茨海默病神经影像倡议）研究自 2004 年起持续开展，目前已划分为 5 个阶段，分别是 ADNI1、ADNIGO、ADNI2、ADNI3 和 ADNI4。当前正依据 ADNI4 方案进行数据收集。需注意，不同阶段的影像检查方案、临床测量方法以及数据的组织与格式可能存在差异。

  • 脑影像数据（包括 MRI [磁共振成像]、PET [正电子发射断层扫描]、病理切片）可通过 “Search and Download → Advanced Image Search” 路径查找。

  • .csv 格式的表格数据可通过 “Study Files listings” 下载。登录 IDA（ADNI 数据访问平台）后，上述两类数据均可在顶部菜单栏的 “Search and Download” 标签页中找到。以下列出部分关键表格在该界面中的具体位置：

    • 数据字典：Study Files → START HERE → Quick Start → Data dictionary

    • 诊断信息：Study Files → Assessments → Diagnosis → Diagnostic Summary

    • 人口统计学数据（含种族、性别等）：Study Files → Subject Characteristics → Subject Demographics → Subject Demographics

    • ApoE4 基因分型数据：Study Files → Biospecimen → Biospecimen Results → ApoE Genotyping – Results

    • 认知评估数据：Study Files → Assessments → Neuropsychological

    • MRI 测量数据：Study Files → Imaging → MR Image Analysis

    • PET 测量数据：Study Files → Imaging → PET Image Analysis

    • 生物体液测量数据：Biospecimen → Biospecimen Results

  • 数据集中使用的主要参与者标识为 PTID（格式为 XXX_S_XXXXX）和 RID。其中，RID 由 PTID 的末尾数字构成，且需去除数字前的所有前导零（例如，若 PTID 为 037_S_4410，对应的 RID 则为 4410）。

  • 在纵向分析中，应使用 VISDATE（访视日期）、EXAMDATE（检查日期）或 SCANDATE（扫描日期）字段确定时间维度（即不同时间点数据的关联）。

  • 需注意，对于 MRI、PET 和生物体液的数值型测量结果，每个.csv 文件均配有对应的.pdf 文档，该文档会详细说明各项测量所采用的具体方法。
